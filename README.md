# The Next Big Thing (v1)

A launch-ready personal/founder website scaffold built with Next.js.

## Pages
- Home
- Spotlight (master's thesis with deeper branching tree)
- Projects (lighter trees for performance)
- Essays
- Contact/About

## Run locally
```bash
npm install
npm run dev
```

## Replace placeholders
- `you@example.com` in `app/contact-about/page.tsx`
- LinkedIn/GitHub URLs in `app/contact-about/page.tsx`
- Spotlight and project tree content in `app/spotlight/page.tsx` and `app/projects/page.tsx`
- Essay placeholders in `app/essays/page.tsx`
